flameplot's documentation!
==============================

``flameplot`` is a Python package for the comparison of high dimensional embeddings using a scale dependent similarity measure.
With flameplot you can quantify the local similarities across two maps or embeddings, such as PCA and t-SNE or with the original high dimensional data.


.. tip::
	`Medium blog: The Similarity between t-SNE, UMAP, PCA, and Other Mappings <https://towardsdatascience.com/the-similarity-between-t-sne-umap-pca-and-other-mappings-c6453b80f303>`_
